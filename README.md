## 📌 Project Overview

서울은 세계에서 가장 고밀도 도시 중 하나입니다. 좁은 땅 위에 역사, 미식, 자연이 층층이 쌓여 있지만, 지상에서는 그 전체를 온전히 경험하기 어렵습니다.

**SKY BOUND**은 UAM(Urban Air Mobility)을 활용해 서울을 하늘에서 경험하는 비행 체험 게임입니다. 플레이어는 원하는 코스를 선택해 비행하고, Google Gemini가 실시간 음성 해설과 대화형 가이드를 제공합니다.

---

## 🎮 Features

### 비행 카테고리
| 카테고리 | 설명 |
|---|---|
| 🏯 랜드마크 | 경복궁, 남산 N타워, 광화문, 롯데타워 등 |
| 🍜 맛집 | 을지로, 홍대, 성수동, 광장시장 등 |
| 🌿 자연경관 | 한강, 북한산, 노들섬, 올림픽공원 등 |

### Gemini 기반 AI 가이드
- 주요 지점 도착 시 **실시간 장소 해설** 자동 제공
- **음성인식(STT)** 으로 자유로운 질문 가능
- **TTS** 기반 자연스러운 음성 응답
- 자체 DB 기반 + Gemini가 콘텐츠 생성 및 보완
